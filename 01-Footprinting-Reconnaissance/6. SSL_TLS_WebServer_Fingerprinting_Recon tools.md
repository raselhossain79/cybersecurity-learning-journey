# 🔐 SSL/TLS & Web Server Fingerprinting — Tools

---

## 1. Certificate Transparency (CT) Log Lookup

**Web**
- [crt.sh](https://crt.sh)
- [Censys](https://search.censys.io)
- [Cert Spotter](https://sslmate.com/certspotter/)
- [Google CT](https://certificate.transparency.dev/)

**CLI**
```bash
amass enum -d example.com
subfinder -d example.com
curl "https://crt.sh/?q=%.example.com&output=json"
```

---

## 2. SSL / TLS Analysis

**Web**
- [Qualys SSL Labs](https://www.ssllabs.com/ssltest/)
- [DigiCert Checker](https://www.digicert.com/help/)
- [SSL Shopper](https://www.sslshopper.com/ssl-checker.html)

**CLI**
```bash
./testssl.sh example.com
sslyze example.com
openssl s_client -connect example.com:443
nmap --script ssl-cert -p 443 example.com
sslscan example.com
```

---

## 3. TLS Version & Cipher Scan

**Web**
- [Qualys SSL Labs](https://www.ssllabs.com/ssltest/)
- [Pentest Tools](https://pentest-tools.com/)

**CLI**
```bash
./testssl.sh --cipher-per-proto example.com
nmap --script ssl-enum-ciphers -p 443 example.com
sslyze --sslv2 --sslv3 --tlsv1 --tlsv1_1 example.com
sslscan --show-ciphers example.com
```

---

## 4. TLS Vulnerability Scan

**Web**
- [Qualys SSL Labs](https://www.ssllabs.com/ssltest/)
- [ImmuniWeb](https://www.immuniweb.com/ssl/)
- [Pentest Tools](https://pentest-tools.com/)

**CLI**
```bash
./testssl.sh --vulnerable example.com
sslyze --heartbleed --robot example.com
nmap --script ssl-heartbleed -p 443 example.com
```

## 5. JA3 / TLS Fingerprinting

**Web**
- [ja3er.com](https://ja3er.com)
- [TLS Fingerprint](https://tlsfingerprint.io)

**CLI**
```bash
python ja3.py -j capture.pcap
zeek -r capture.pcap
tshark -r file.pcap -Y tls.handshake
```

## 6. HTTP Header Analysis

**Web**
- [Security Headers](https://securityheaders.com)
- [Mozilla Observatory](https://observatory.mozilla.org)

**CLI**
```bash
curl -I -s https://example.com
httpx -u example.com -title -server -status-code
nikto -h example.com
```

## 7. Web Server Fingerprinting

**Web**
- [Netcraft](https://sitereport.netcraft.com)
- [Shodan](https://www.shodan.io)
- [Censys](https://search.censys.io)

**CLI**
```bash
whatweb https://example.com
httpx -u example.com -server -tech-detect
nmap -sV -p 80,443 example.com
wafw00f https://example.com
```
